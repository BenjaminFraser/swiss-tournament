Welcome

The Goody Basket is a reliable, convenient and simple interface through which users can create their very own item category, which contains a set of listed products. There is full CRUD functionality for viewing, editing and deleting individual items, along with whole categories.
View the category list at the listings page to get started.

There is no long and arduous registration requirement for an account with us, simply use your Google+ or Facebook login and you can be on track to becoming a Goody Basket trader!

This platform was build by Benjamin D Fraser using the Flask framework, in conjunction with SQLAlchemy, SQLite, Bootstrap, custom HTML/CSS, JavaScript and Python. Form validation was built using WTForms within Flask.
